Projeto desenvolvido em TypeScript com o objetivo de demonstrar a implementação do padrão de projeto Abstract Factory.

O sistema simula a criação de personagens de um jogo, onde cada fábrica é responsável por criar uma família de objetos relacionados, como armas e armaduras específicas para cada personagem (Guerreiro e Mago).

O padrão Abstract Factory foi utilizado para reduzir acoplamento, facilitar manutenção do código e permitir a criação de novos personagens sem modificar a estrutura principal do sistema.

Tecnologias utilizadas:
- TypeScript
- Node.js
- Nodemon
- Programação Orientada a Objetos (POO)
- Design Pattern: Abstract Factory
